

# Portfolio Pessoal

Bem-vindo ao repositório do meu portfólio pessoal. Este projeto é um site que destaca minha jornada acadêmica, projetos e habilidades.

## Visão Geral

Este portfólio foi desenvolvido usando tecnologias web modernas, incluindo React.js para a construção da interface e outros frameworks e bibliotecas relacionados.
